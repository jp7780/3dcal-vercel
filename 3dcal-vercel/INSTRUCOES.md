# 🚀 Deploy 3dcal no Vercel

## 📁 Arquivos Essenciais (JÁ PRONTOS!)

✅ **vercel-app.py** - Aplicação Flask completa e independente  
✅ **vercel.json** - Configuração do Vercel  
✅ **requirements.txt** - Dependências Python  
✅ **README.md** - Documentação  
✅ **.gitignore** - Arquivos ignorados  

## 🎯 Upload para GitHub

### 1. Criar Repositório
- Acesse github.com → New repository
- Nome: `3dcal-web`
- Public
- Create repository

### 2. Upload dos Arquivos
- Clique em "uploading an existing file"
- Arraste os 5 arquivos da pasta `3dcal-web`
- Commit changes

### 3. Deploy no Vercel
- Acesse vercel.com
- Login com GitHub
- New Project → Importar `3dcal-web`
- Framework: Python
- Build Command: (deixe em branco)
- Install Command: `pip install -r requirements.txt`
- Deploy!

### 4. Configurar Domínio
- Após deploy: Settings → Domains
- Adicionar: `calcular3d.com`
